# README

1、这是一个利用VuePress搭建的静态网站生成器，我将它简单改了一下，在默认主题基础上换了页面布局，作为我的个人博客

2、VuePress是以Vue驱动的静态站点生成工具，特点是以markdown为中心，同时可以在markdown中使用vue组件，也可以通过vue开发自定义主题

3、博客目前处于初始化阶段，暂时只有简单的页面展示，未来会渐进式增加一些实用功能
